flutter 命令失效的问题

在之后的使用过程中发现，flutter 命令会失效，并且原先 Export 到 $PATH 中的 flutter 路径变量莫名消失。

这里提供解决方案，解决方案应该是不唯一的。

  1. 终端执行 sudo touch /etc/paths.d/flutter ，在 $PATH 目标路径下新增变量文件。
  2. 终端执行 sudo vim /etc/paths.d/flutter ，打开新建的 flutter 文件进行编辑。
  3. 在文件中输入 /Users/chanjason/flutter/bin (这里是替换成你系统中 flutter 安装的路径)。
  4. 重启终端。
